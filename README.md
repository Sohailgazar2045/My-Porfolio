# Muhammad Sohail - Portfolio Website

A modern, responsive portfolio website built with React and CSS.

## Features

- Responsive design (mobile, tablet, desktop)
- Clean, professional UI with smooth animations
- Dynamic content loading from JSON file
- SEO-friendly structure
- Accessible navigation

## Setup Instructions

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm start`
4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser

## Customization

1. Update the `data.json` file with your personal information
2. Replace the profile image placeholder with your actual image
3. Add your project screenshots and update project links
4. Modify colors in CSS files to match your preferred style

## Deployment

Build the project: `npm run build`

The build folder contains production-ready files that can be deployed to any static hosting service (Netlify, Vercel, GitHub Pages, etc.).
